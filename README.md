# Work with API
 create a web page with the current exchange rate of the Belarusian ruble in relation to all the foreign currencies established by the National Bank of the Republic of Belarus. The course should always be up-to-date at the time the web page is opened.

You can find the official exchange rate of the Belarusian ruble in relation to foreign currencies here (https://www.nbrb.by/statistics/rates/ratesdaily.asp).
